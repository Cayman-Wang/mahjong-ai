## 5. 仍需跟踪（非阻塞）

1. RLlib 新版本兼容性：当前训练可跑，但 `build` / `compute_single_action` 仍有 deprecation 警告，后续建议迁移到 RLModule 新接口。
2. 默认 `python` 解释器下 RL 相关测试会跳过；若要把 RL 测试纳入常规回归，需要统一训练解释器与依赖环境（建议 Python 3.10/3.11）。
